# wecube-common-ui

## 安装
npm install wecube-common-ui --save

## 引入
import commonUI from 'wecube-common-ui'
import 'wecube-common-ui/lib/wecube-common-ui.css'

## 全局注册
Vue.use(commonUI)

## babel.config.js添加配置
sourceType: 'unambiguous'

## 发布步骤
1.cd common-ui目录
2.npm addUser wanghao7717
3.npm publish
